## ⚠️ ما يحتاج إلى عمل

### 1. التحقق من رفع الصورة الشخصية
**الحالة**: الكود موجود وصحيح ✅
**المطلوب**: إعداد Supabase Storage

**الخطوات**:
1. إنشاء bucket "profiles" في Supabase Storage
2. إضافة policies للصلاحيات (موجودة في AVATAR_UPLOAD_VERIFICATION_AR.md)
3. اختبار رفع الصورة

---

### 2. تحسين بطاقة الكورس
**الموقع**: `courses_page.dart` → `_buildCourseCard()`

**التحسينات المقترحة**:
```dart
- إضافة gradient للصورة
- تحسين الظلال والألوان
- إضافة badge للمستوى (مبتدئ/متوسط/متقدم)
- تحسين عرض التقييم والطلاب
- إضافة أيقونات أفضل
- تحسين المسافات والتنسيق
```

---

### 3. تحسين صفحة تفاصيل الكورس
**الموقع**: `course_detail_page.dart`

#### أ. تبويب الوصف
```dart
- تنسيق النص بشكل أفضل
- إضافة أيقونات للمعلومات (المدة، المستوى، اللغة)
- عرض معلومات المدرب بشكل أجمل
- إضافة قسم "ماذا ستتعلم"
```

#### ب. تبويب المحتوى
```dart
- تصميم ExpansionTile أفضل للوحدات
- إضافة أيقونات للدروس (فيديو، مقال، اختبار)
- عرض مدة كل درس
- إضافة مؤشر التقدم للطلاب المسجلين
```

#### ج. تبويب التقييمات
```dart
- تصميم بطاقات التقييمات
- عرض النجوم بشكل جميل
- صورة المستخدم واسمه
- تاريخ التقييم
- زر "مفيد" للتقييمات
```

---

### 4. تحسين صفحة الإعدادات
**الموقع**: `settings_screen.dart`

**التحسينات المقترحة**:
```dart
- تقسيم الإعدادات إلى أقسام (الحساب، الإشعارات، المظهر، الخصوصية)
- استخدام Switch widgets للخيارات
- تصميم أنيق للبطاقات
- أيقونات محسّنة
- ألوان متناسقة
```
